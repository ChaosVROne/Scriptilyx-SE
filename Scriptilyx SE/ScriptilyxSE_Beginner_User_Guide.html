1. Start here

Scriptilyx SE is a visual builder for Space Engineers programmable block scripts. Add nodes, connect them, generate code, and paste it into a programmable block.

Use nodes
Each node does one job, like turning on a block, checking a sensor, setting a sorter filter, or writing to an LCD.
Connect ports
Connect Next, True, False, Loop, Done, and custom outputs to other node inputs.
Generate Serious Script
Use this for the script you paste into Space Engineers.
Test small
Build one small test graph first. Then add more automation.

3. Basic graph workflow

  1. Add one Start node.
  2. Add action/check nodes.
  3. Connect Start.Next to the next node In.
  4. Fill in node properties on the right panel.
  5. Connect both True and False on condition nodes.
  6. Click Check.
  7. Click Generate Serious Script.
  8. Copy or save the script.
  9. Paste into Space Engineers and click Check Code.
Best test: Start → Set LCD Text → Stop Script.

5. Script Preview and Generate Serious Script

Use Generate Serious Script before copying, saving, or pasting into Space Engineers. That is the generator intended for in-game testing.

  1. Build or open a graph.
  2. Click Generate Serious Script.
  3. Review Script Preview.
  4. Click Copy or Save.
  5. Paste into Space Engineers.

6. Script Folders

  1. Open Script Folders tab.

  3. Select or create a folder.
  4. Generate Serious Script.

8. Scanner script

The scanner script is used in Space Engineers to discover exact terminal actions and properties from real blocks.

Scanner workflow

  1. Paste the scanner script into a programmable block.
  2. Run it on the grid with the blocks you want to scan.
  3. Copy the output from Custom Data/output.
  4. Save it as an API dump file.
  5. Use Import API Dump in Scriptilyx SE.

Use it for modded blocks, exact ActionId values, exact PropertyId values, AI checks, Event Controller checks, and advanced terminal controls.

9. Import API Dump

Import API Dump reads scanner output and helps create nodes or use generic terminal nodes.

Dump dataUse node
Terminal actionApply Any Terminal Action
Bool propertySet Any Bool Property
Float propertySet Any Float Property
String propertySet Any Text Property
Int propertySet Any Int Property
  1. Click Import API Dump.
  2. Select the scanner dump file.
  3. Review found blocks/actions/properties.
  4. Create or import the nodes you need.
  5. Test one imported node at a time.

12. Pick Item ID and Item Database

Use Pick Item ID on sorter item nodes instead of typing long item IDs.

  3. Search for the item.
  4. Double-click/select the item.
