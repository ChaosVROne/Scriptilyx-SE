# Scriptilyx-SE
Scriptilyx-SE is a desktop tool for building Space Engineers programmable block scripts using a visual node-based editor.

This repository is used for public downloads, documentation, and release files.

## Download

Go to the **Releases** section and download the latest `.zip` file.

## Included Files

- `Scriptilyx SE.exe` - Main application
- `NodeLibrary.json` - Node database
- Runtime files required by the app
- Beginner user guide: ScriptilyxSE_Beginner_User_Guide.html
